About the Application

The Tomato Leaf Disease Predictor is a flask web application which classifies a plant/leaf image into 10 categories. Tomato mosaic virus, Early blight, Septoria leaf spot, Bacterial spot, Target Spot, Spider mites Two spotted spider mite, Tomato Yellow Leaf Curl Virus, Late blight, Healthy, and Leaf Mold. The code is written in Python 3.10 and makes use of Keras and Tensorflow libraries in developing an InceptionV3 based image classification web application.

Technology used

    Tensorflow : used to train and build the model.
    Numpy : used for performing mathematical and logical operations on Arrays.
    Pandas : used for data cleaning and analysis.
    Open Cv : used to reading or re-sizing an image
    Matplotlib : used to visualization of data
